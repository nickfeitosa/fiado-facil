# Fiado Fácil

## Sobre o Projeto

O Fiado Fácil é uma plataforma digital desenvolvida para auxiliar microempreendedores e pequenos comerciantes no controle de vendas fiadas.

A solução foi criada como atividade da disciplina de Transformação Digital, alinhada ao Objetivo de Desenvolvimento Sustentável (ODS) 8 – Trabalho Decente e Crescimento Econômico.

O sistema substitui controles manuais realizados em cadernos e anotações informais, permitindo maior organização, transparência e acompanhamento das vendas realizadas a prazo.

---

## Funcionalidades

* Cadastro de clientes
* Registro de compras fiadas
* Cálculo automático do saldo devedor
* Dashboard com indicadores operacionais
* Histórico de compras recentes
* Geração automática de comprovante copiável para envio ao cliente
* API documentada automaticamente com Swagger

## Objetivo de Desenvolvimento Sustentável

### ODS 8 – Trabalho Decente e Crescimento Econômico

A solução busca promover a digitalização acessível de pequenos negócios, reduzindo perdas financeiras causadas por controles manuais e contribuindo para a melhoria da gestão operacional dos microempreendedores.
